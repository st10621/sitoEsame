Progetto 1: Sistema di Gestione Biblioteca

Applicazione Java per la gestione di una biblioteca con funzionalità CRUD per libri, utenti e prestiti. Il sistema include un'interfaccia grafica intuitiva e un database relazionale MySQL per la persistenza dei dati.

Codice Sorgente

Visualizza su GitHub
// Classe principale per la gestione dei libri
public class Libro {
  private String titolo;
  private String autore;
  private String isbn;
  private boolean disponibile;

  public Libro(String titolo, String autore, String isbn) {
    this.titolo = titolo;
    this.autore = autore;
    this.isbn = isbn;
    this.disponibile = true;
  }
// Metodi getter e setter omessi per brevità
}

Commenti sul Codice

Architettura MVC: Il progetto segue il pattern Model-View-Controller per separare la logica di business dall'interfaccia utente.

Gestione database: Utilizzo di JDBC per le operazioni CRUD su database MySQL con transazioni per garantire l'integrità dei dati.

Eccezioni personalizzate: Implementazione di eccezioni specifiche per gestire errori come "LibroNonDisponibileException" e "UtenteNonTrovatoException".

Progetto 2: Simulatore di Rete

Applicazione Python per simulare il funzionamento di una rete di computer, inclusi router, switch e dispositivi client. Implementazione di algoritmi di routing e gestione del traffico di rete.

Codice Sorgente

Visualizza su GitLab
# Classe per rappresentare un router
class Router:
  def __init__(self, name):
    self.name = name
    self.routing_table = {}
    self.interfaces = []

  def add_route(self, destination, next_hop, interface):
    self.routing_table[destination] = (next_hop, interface)

  def forward_packet(self, packet):
    if packet.destination in self.routing_table:
      next_hop, interface = self.routing_table[packet.destination]
      interface.send(packet)
    else:
      print(f"Destination {packet.destination} not found in routing table")

Commenti sul Codice

Algoritmi di routing: Implementazione di OSPF e RIP per la determinazione dei percorsi ottimali nella rete.

Simulazione del traffico: Generazione di pacchetti casuali per testare le prestazioni della rete in diverse condizioni.

Visualizzazione: Utilizzo di Matplotlib per generare grafici delle prestazioni in tempo reale.

Progetto 3: App Meteo per Android

Applicazione mobile per Android che mostra le previsioni meteo basandosi sulla posizione dell'utente. Integrazione con API meteorologiche e gestione offline dei dati recenti.

Codice Sorgente

Visualizza su Bitbucket
// Funzione per ottenere i dati meteo
private fun fetchWeatherData(location: Location) {
  val call = weatherService.getWeather(
    location.latitude,
    location.longitude,
    "metric",
    API_KEY
  )

  call.enqueue(object : Callback<WeatherResponse> {
    override fun onResponse(
      call: Call<WeatherResponse>,
      response: Response<WeatherResponse>
    ) {
      if (response.isSuccessful) {
        val weatherData = response.body()
        updateUI(weatherData)
        saveToDatabase(weatherData)
      }
    }
  })
}

Commenti sul Codice

Architettura MVVM: Utilizzo del pattern Model-View-ViewModel con LiveData per un aggiornamento reattivo dell'interfaccia.

Gestione offline: Implementazione di Room Database per salvare localmente le previsioni e mostrarle quando non c'è connessione.

Permessi: Gestione avanzata dei permessi per l'accesso alla posizione con fallback alla ricerca manuale.
