Access List (ACL)

Le Access Control List sono strumenti utilizzati nei router per filtrare il traffico in base a criteri specifici (IP, porta, protocollo). Possono essere standard o estese e servono per migliorare la sicurezza e la gestione del traffico di rete.

NAT e PAT

Il NAT (Network Address Translation) permette di tradurre indirizzi IP privati in uno pubblico. Il PAT (Port Address Translation) estende il NAT permettendo a più dispositivi interni di condividere un singolo IP pubblico usando porte diverse.

Reti Wi-Fi

Le reti Wi-Fi permettono la connessione senza fili. È importante configurare la sicurezza tramite WPA2/WPA3, evitare SSID aperti, e controllare l'accesso tramite MAC address o captive portal.
